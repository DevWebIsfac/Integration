# Exercice 1

### Énoncé du client

&laquo; Je souhaite que tu me créés une page web pour présenter cet article que je viens d'écrire à propos du language CSS.

Je te donne deux fichiers texte contenant les textes que je viens de terminer : un pour l'intro et un autre pour le reste. Comme c'est un peu long, j'aimerai qu'entre l'intro et le reste du texte tu fasses un sommaire cliquable qui permette à l'internaute d'accéder directement à la rubrique qui l'intéresse. Un peu comme sur Wikipedia. &raquo;

### Ce que vous devez faire

1. Vous devez fournir un fichier compressé (zip ou rar) du répertoire de votre site, ce répertoire devra s'appeler `[prenom]-article-css` et me le transmettre par mail
2. Votre répertoire devra comprendre deux fichiers : `index.html` et `article.css`
3. Votre page devra être valide selon le W3C
4. Votre page aura un titre précisant le contenu de votre page
5. Votre code html et css devra être correctement indenté
6. L'apparence de l'introduction devra ête différente du reste du texte
7. Le menu devra être avoir une couleur de fond différente du blanc

### Conseils

1. Un sommaire/menu/navigation représente une arborescence. En HTML, on utilise des listes non-ordonnées pour les représenter.
2. Il est possible d'imbriquer plusieures listes pour obtenir plusieurs niveaux.
3. Pour faire un lien interne vers une balise possédant l'attribut `id="pomme"` on écrira le lien comme ceci `<a href="#pomme">Mon lien</a>`
4. Définissez la propriété css `float` au menu et mettez comme valeur `left`
