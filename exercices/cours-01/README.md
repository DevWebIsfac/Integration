# Intégration - Cours 1 - Les bases

### La présentation

Vous pouvez consultez les slides de la présentation à cette adresse <a href="https://devwebisfac.github.io/Integration/slides/cours-01.html" target="_blank">https://devwebisfac.github.io/Integration/slides/cours-01.html</a>

### Notions abordées lors du cours

- Petit historique du web
- Les bases du HTML :
    + gabarit html 5 (le squelette)
    + les balises orphelines / par paires
    + la syntaxe des balises
    + les attributs
    + l'imbrication des balises
    + l'indentation
- Les premières balises HTML :
    + mise en forme du texte
        * les titres (h1, h2, h3, h4, h5, h6)
        * les paragraphes (p)
        * strong (important) et em (emphase)
        * les listes ordonnées (ol) et non-ordonnées (ul)
        * les liens
    + insérer une image (img) et son texte alternatif
- Les bases du CSS
    + Premières mises en forme CSS :
        * css inline : l'attribut `style`
        * dans le head (balise `<style>`
        * dans un fichier css externe
    + La syntaxe CSS :
        * sélecteur de balise
        * propriété
        * valeur
    + quelques propriétés :
        * couleur de texte `color`
        * couleur de fond `background-color`
        * police `font-family`
        * alignement du texte `text-align`
        * marges externes `margin`
        * marges internes `padding`
- premiers pas avec les outils de développement de Google Chrome (clic-droit Inspecter)
