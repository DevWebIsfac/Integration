Tableaux : fusion de cellules

Consignes

  • Toutes les cellules de fond turquoise ou noir sont des cellules d'en-tête <th>
  • Le code couleur de la couleur turquoise est teal
  • Tentez d'avoir un code CSS le plus concis possible. Pour information, le code CSS du corrigé fait 26 lignes.
  • Pour obtenir un É si vous ne savez pas comment le taper sur votre clavier, utilisez le code &Eacute;
  • Pour les cellules vides, vous devez insérer un espace insécable. Un espace blanc dans le code n'étant pas interprété, utilisez le code &nbsp; pour None Breakable SPace
  • Tous les textes sont saisis en minuscule dans le code HTML et mis en majuscule en CSS
  • Attention au respect de la typographie (type serif/sans-serif, graisse de la police)
  • Pour appliquer une ombre portée à vos tableau, utiliser le code css suivant : box-shadow: 2px 4px 9px rgba(0,0,0,0.2);
  • Penser à valider votre code sur le site du W3C
