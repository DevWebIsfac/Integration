# Cours 03 - Modèle de boîte et positionnement CSS

## Présentation

Les slides sont disponible à <a href="https://devwebisfac.github.io/Integration/slides/cours-03.html" target="_blank">cette adresse</a>

- Exercice 1 : **dimensions et modèle de boîte**
    + width,
    + height,
    + padding,
    + margin,
    + border
- Exercice 2 : **positionnement**
    + position relative `position: relative;`,
    + position absolue `position: absolute;`,
    + position fixe `position: fixed;`,
- Exercice 3 : **centrage horizontal** et **box-sizing**
    + on centre horizontalement grâce à `margin-left: auto;` et `margin-right: auto;`
- Exercice 4 : **gabarits de page html**
    + contenu fixe et centré
    + *idem* avec : un header, une sidebar, un contenu et un footer
    + *idem* avec deux sidebars
    + design fluide avec une largeur max `max-width: 960px;`

## Exercice 5 : **création d'un site 4 pages avec un menu**

### Structure du projet

- `votre_dossier/
    + `index.html` : l'accueil
    + `page1.html` : la page n°1
    + `page2.html` : la page n°2
    + `page3.html` : la page n°3
    + `style.css` : la feuille de style

### Consignes

> L'exercice est a terminer sur votre journée d'assimilation du 5 octobre :
> **Pensez à vérifier la validité de votre code avec le validateur du W3C**

- chacune des 4 pages doit contenir : un header et une zone de contenu avec un titre spécifique,
- le header doit occuper toute la largeur de l'écran,
- le contenu doit avoir une largeur maximum (1040px par exemple),
- le menu doit être dans le header et représenté comme ceci :
    + chaque lien à l'apparence d'un bouton,
    + tous les boutons sont sur la même ligne,
    + le bouton de la page sur laquelle on se trouve doit être reconnaissable visuellement
