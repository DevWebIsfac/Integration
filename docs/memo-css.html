Explications

  • <longeur> :
    • <longeur> doit être remplacé par une longueur absolue ou relative (px, em, rem, …).
    • Le nombre est immédiatement suivi de l'unité, sans espace. Par exemple 1.6em et pas 1.6 em
    • La valeur 0 ne doit pas comporter d'unité
  • <pourcentage> :
    • <pourcentage> doit être remplacé par un pourcentage.
    • Le nombre est immédiatement suivi de symbole %, sans espace. Par exemple 15% et pas 15 %
    • La valeur 0 ne doit pas comporter d'unité
