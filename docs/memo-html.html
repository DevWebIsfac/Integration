Balise Définition et Attributs Exemple
a Hyperlien
  • href: cible URL (absolue ou relative)
    Valeurs possibles :
    • une url absolue : href="http://developper.mozilla.org"
    • une url relative : href="page-contact.html"
    • un fragment url : href="#form" qui enverra sur l'élément de la même page avec l'identifiant id="form"
    • mailto avec adresse email : href="mailto:john.doe@test.com". Le clic sur le lien ouvre le client de messagerie par défaut et prépare un nouveau message avec le destinataire pré-rempli.
  • target : définie où afficher la ressource. Par exemple target="_blank", permet d'ouvrir le lien dans un nouvel onglet. 
<a href="newsletter.php" target="_blank">
  Inscription newsletter
</a>

div Balise universelle de type block
Cela signifie division de page. Sa seul fonction est d'organiser le contenu en regroupant des éléments et leur appliquer un style (grâce à un id ou une class). Tant que l'on ne change pas le style, la div n'a aucune fonction ou effet sur la page. 
<div></div>
span Balise universelle de type inline
Même principe que pour la div mais de type inline 
<span></span>

Formulaires

Balise Définition et Attributs Exemple
form Formulaire
  • method : définie la methode d'envoi du formulaire
    Valeurs possibles : action | post
  • action : url du fichier à qui soumettre le formulaire
    Valeurs possibles <url> 
<form action="traitement.php" method="post">
  <!-- Placer tous les champs ici -->
</form>
input Champ de saisie
  • type : définie le type de champ
    Valeurs possibles : button | checkbox | email | file | hidden | password | radio | tel | text
  • checked : lorsque le champ est de type checkbox ou radio, permet de cocher la case à cocher ou l'option
    Valeurs possibles : checked
  • name : nom de contrôle envoyé avec les données du formulaire
    Valeur possible : une chaîne de caractère sans espace
  • placeholder : indication fournie à l'utilisateur pour saisir ses données. Par exemple, sur un champ texte, le placeholder remplira le champ par un texte grisé qui sert d'exemple à l'utilisateur 
<input type="checkbox" name="newsletter" checked="checked" />
<input type="text" name="ville" placeholder="Saisissez le nom de votre ville de naissance" />
label Étiquette de libellé d'un champ de saisie
  • for : identifiant id du champ correspondant
 
<!-- Sans attribut for, le champ doit être à l'intérieur -->
<label>
  Prénom :
  <input type="text" name="prenom" />
</label>

<!-- Avec attribut for, le champ peut être à l'extérieur -->
<label for="f_firstname">Prénom :</label>
<input type="text" id="f_firstname" name="prenom" />
option Élément de contrôle d'un élément select
  • disabled : booleén qui indique si l'on peut sélectionner cette option
  • selected : booleén qui indique si l'option est sélectionnée à l'état initial
  • value : valeur que l'on souhaite envoyée au formulaire. Si non défini, c'est le contenu de la balise qui est envoyé
 
<!-- Sans attribut value, c'est le contenu (vert ou rouge) qui est envoyé -->
<select name="couleur">
  <option>vert</option>
  <option>rouge</option>
</select>

<!-- Avec attribut value, c'est lui (green ou red) qui est envoyé au formulaire -->
<select name="couleur">
  <option value="green">vert</option>
  <option value="red">rouge</option>
</select>
select Liste déroulante d'options
  • disabled : booléen indiquant si on peut intéragir avec la liste. Si non, elle apparait en grisé
  • multiple : booléen indiquant si l'on peut choisir plusieurs options
  • name : nom du contrôle qui permettra de récupérer les données côté traitement php
 
<select name="couleur" disabled="true">
  <option>vert</option>
  <option>rouge</option>
  <option>orange</option>
</select>
textarea Zone de texte multiligne
  • disabled : booléen indiquant si on peut intéragir avec la liste. Si non, elle apparait en grisé
  • name : nom du contrôle qui permettra de récupérer les données côté traitement php
  • placeholder : texte indicatif affiché avant la saisie par l'utilisateur
 
<textarea name="message" placeholder="Écrivez votre message ici">
</textarea>
fieldset Groupe de champs et labels au sein d'un formulaire
  • disabled : booléen indiquant si on peut intéragir avec la liste. Si non, elle apparait en grisé
Le nom du groupe est déterminé par le premier élément enfant legend 
<form action="index.php" method="post">
  <fieldset>
    <legend>Identité</legend>
    <input type="text" name="prenom" />>
    <input type="text" name="nom" />>
  </fieldset>
</form>
legend Légende d'un élément fieldset 
<!-- Voir exemple de `fieldset ci-dessus -->
optgroup Groupe d'option dans un select
  • disabled : booléen indiquant si on peut intéragir avec la liste. Si non, elle apparait en grisé
  • label : nom du groupe d'option. Cet attribut est obligatoire
 
<select name="categorie">
  <optgroup label="Autos :">
  	<option>Berline</option>
  	<option>Break</option>
  	<option>Monospace</option>
  	<option>Coupé / Cabriolet</option>
  </optgroup>
  <optgroup label="Motos :">
  	<option>Routière</option>
  	<option>Tous terrains</option>
  	<option>Circuit</option>
  </optgroup>
</select>
