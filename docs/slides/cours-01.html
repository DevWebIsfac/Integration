Exercice 1

Un fichier avec l'extension .HTML

  • Créez un répertoire nommé *Intégration*
  • Dans ce répertoire, créez un répertoire nommé *01-Les-bases*
  • Créez dans ce répertoire un fichier texte et renommez le *exercice-01.txt*
  • Éditez ce fichier avec l'éditeur de texte de votre choix (choisissez un éditeur de texte brut, n'utilisez pas Word mais plutôt le bloc-note)
  • Écrivez le texte *Hello World* et enregistrez le fichier
  • Retournez dans l'explorateur de fichier, renommez ce fichier *exercice-01.html* et double-cliquez sur le fichier **exercice-01.html**

Que constatez-vous ?

  • L'ordinateur ouvre le fichier automatiquement dans voter navigateur web par défaut (grâce à l'extension .html)
  • Le navigateur affiche une page blanche avec le texte "Hello World" en noir
  • L'onglet indique le nom du fichier
  • La barre d'adresse indique l'adresse locale du fichier

Exercice 2

Préparer un squelette réutilisable

  • Dans notre répertoire *01-Les-bases*, créer un nouveau fichier nommé *squelette-html5.html*
  • Éditez ce fichier avec le bloc-notes
  • Définissez le type de document html 5
  • Spécifiez que le fichier utilise la langue française
  • Créez une en-tête et un corp de document
  • Définissez l'encodage de caractère à utf-8'
  • Donner un titre à votre page
